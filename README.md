## Структура проекта

- `api/` — API сервис
- `connector/` — Kafka consumer, отправка платежей в эквайер
- `acquirer_simulator/` — тестовый эквайер
- `infra/` — инфраструктура Docker Compose
- `migrations/` — SQL миграции
